# Коллекция Полезных Утилит на GitHub

Этот репозиторий представляет собой курируемый список полезных утилит и инструментов с открытым исходным кодом, доступных на GitHub.

## Инструменты командной строки

*   **[GitHub CLI](https://github.com/cli/cli)** - `gh` — это официальный инструмент командной строки для GitHub. Он переносит пул-реквесты, ишью и другие концепции GitHub в терминал рядом с тем местом, где вы уже работаете с git и вашим кодом.
